Smart Detect — Auto-find Dimensions

This feature automatically scans your PDF for dimension text and places balloons next to each detected dimension.

Works best with: Vector PDFs exported from CAD software (SolidWorks, AutoCAD, Creo, Inventor, NX). These have embedded text data that can be read directly.
Does NOT work with: Scanned/photographed drawings (raster PDFs). These are just images with no text data. Use manual ballooning instead.

How to tell: If you can select text in your PDF using a regular PDF viewer (Adobe, Chrome), it's a vector PDF and Smart Detect will work. If you can't select any text, it's scanned.

Detected dimensions will be placed as balloons. You can delete wrong ones and add missed ones manually afterward.
